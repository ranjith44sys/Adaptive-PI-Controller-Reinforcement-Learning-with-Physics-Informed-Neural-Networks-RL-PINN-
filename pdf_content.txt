 
Caterpillar: Confidential Green  Gain Tuning for PI Controllers Using Machine Learning  
 
Background  
In modern control systems, achieving optimal performance under varying operating 
conditions and disturbances is a critical challenge. Traditional PI controllers often 
require manual tuning of proportional and integral gains, which can be time -consuming 
and suboptimal when system dynamics change. Machine Learning (ML) offers a 
promising approach to adaptively tune controller gains based on real -time feedback 
and system behaviour.  
 
Objective  
Design and implement an ML -based solution that dynamically adjusts the PI controller 
gains to minimize error and improve system stability for a given plant model. The gain 
correction factor is a multiplication on the existing proportional and integral gains 
provided with the controller. The gain correction factor depends on the error (The 
difference between desired and actual value) and the imposed disturbance. The ML 
model should learn from system feedback , requirement parameters and disturbances 
to provide gain correction factors that enhance controller performance.  
 
System Overview  
The control loop consists of:  
• Error Block : Computes the difference between desired value and actual value.  
• PI Controller : Generates control output based on error and tuned gains.  
• Plant Model : Represents the system dynamics influenced by parameters (B, C, 
D) and disturbance (A).  
• ML Model : Can use error, Disturbance A, requirement parameters (defined in 
Requirements figure), plant defining parameters (parameter B, C, &D) and actual 
value feedback, and outputs gain correction factors for the PI controller.  The gain 
correction factor should only be dependent on the error and the Disturbance (A).
